# Mosquito Bioinformatics Projects

## Overview
This repository contains applied bioinformatics projects focused on mosquito vectors and public health research.

## Projects

### 1. Species Identification (BLAST)
- Input: DNA sequence
- Output: Mosquito species identification

### 2. Insecticide Resistance Analysis
- Focus: kdr mutations
- Tools: NGS, SAMtools

### 3. AI-Based Prediction Model
- Predict mosquito density using climate data

## Tools Used
- Python (Biopython)
- R (ggplot2)
- BLAST (NCBI)
